# Webhook Processor

Асинхронный сервис для приёма и обработки вебхуков.

## Технологии
- **FastAPI** – веб-фреймворк
- **MongoDB** – хранение событий
- **Redis** – очередь задач
- **Docker / Docker Compose** – контейнеризация
- **GitHub Actions** – CI/CD
